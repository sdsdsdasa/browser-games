# Browser Games

A collection of retro-style browser games built as single self-contained HTML files. No dependencies, no build step — just open in any modern browser.

## Games

### Top-Down Shooter (`shooter.html`)

A retro-style 2D top-down shooter built entirely with the Canvas 2D API (no external assets).

**How to play:**
- **Arrow keys** — Move (8-directional)
- **Mouse** — Aim
- **Left click** — Shoot

**Features:**
- 4 enemy types: Grunt, Runner, Tank, Shooter
- 5 levels with scaling difficulty (endless after level 5)
- Wave-based spawning from screen edges
- Particle explosions, screen shake, muzzle flash
- HP bar, lives system, score, high score (localStorage)
- Web Audio beep sound effects

---

### Tic-Tac-Toe (`tictactoe.html`)

Classic two-player tic-tac-toe in the browser.

---

## Running

Open any `.html` file directly in Chrome or Firefox — no server needed.
